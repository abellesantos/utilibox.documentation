# Generating NTO

1. From the Analysis Tab, click on Network Tariff Optimisation page.
2. On the top right hand corner of the page, click on _+Generate Report._
3.  Fill in the details on the _Network Tariff Optimisation Report Generation_ modal.

    * Description
    * Utilities to Include
    * Utilities to Exclude
    * State
    * Baseline Start Date - default to previous fiscal year start (can be edited)
    * Baseline End Date - default to previoud fiscal year end (can be edited)
    * Forecast Start Date - default to current fiscal year start (can be edited)
    * Forecast End Date - default to current fiscal year end (can be edited)

    \*\*Option to exclude Non-SLA sites is available to be selected as needed
4. Click on _"Generate Report"_ icon to start the generation process.
5. A pop-up window will appear to confirm if user would want to proceed with the generation
6. Click OK (or Cancel if needed), a pop-up will appear indicating that the report will be generated and Dashboard will automatically refresh once done.
7. A new Dashboard will show ready for review/analysis.
8. Report can be exported in excel as needed by clicking on "Export Report" at the top right hand corner of the page.
