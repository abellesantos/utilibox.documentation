# Settings

On this tab is where a scenario's settings can be edited/updated.

1.  Select the scenario to edit/update.

    Sections and Fields that can be updated are the following:

    Scenario Settings:

    1. Scenario Name
    2. Emission Calculation Method
    3. Baseline Year
    4. Decarbonisation Of Grid

    Target Settings:

    1. Year
    2. Emissions Reduction %

    Virtual Site Closure Settings:

    1. Utility
    2. Closure Date
2. Once all necessary fields are updated/added for each section, click Save.
3. To show all changes, click on Refresh Scenario.
